#Construction Management System

A web-based application to manage construction projects, clients, financial transactions, daily work logs, and site documents efficiently.

## Features

* 👤 Client Management
* 🏗️ Project Tracking
* 💰 Income & Expense Tracking
* 📅 Daily Work Logs
* 📂 Document Management (Permits, Tax, Plans)

## Tech Stack

* Java (Spring Boot)
* PostgreSQL
* Thymeleaf
* Maven

## Status

🚧 In Development

## Goal

To build a real-world system for managing construction workflows in a structured and scalable way.
